# Aula Introdução ao Desenvolvimento de Software

## Pessoas envolvidas no desenvolvimento de um software:



- **UX Design** - Pessoa que visa melhorar a visibilidade e até mesmo a satisfação proporcionada na interação. É quem monta um "esqueleto" do software, por exemplo, onde estará o botão para logar, acessar o menu, etc.  Esse esboço pode ser feito de forma manual, ou através de alguma ferramenta de wireframe. 
- **UI Design** - Vê e utiliza o produto. É quem trata das cores, tipografia, micro-interações e estilos, sempre visando a interface do produto, ou seja, visando a persona que o utilizará. 
- **Front-End**: Programa a parte visual de um site ou aplicativo, ou seja, aquilo que conseguimos interagir. É o responsável por desenvolver, por meio de código, uma interface gráfica, normalmente com as tecnologias base da web (HTML, CSS e Javascript). Ex: Ao clicar em um botão de login em um site, o frontend codifica para o redirecionamento, que é requisitado no servidor, a partir daí, o retorno se existe ou não é feito pelo servidor, no backend. 

​		>_Mas onde esses códigos são criados?_ 

​		Os códigos são criados através de um ambiente de desenvolvimento de softwares, ou *IDE* (Integrated Development Environment). As chamadas IDEs são softwares para criar aplicações que combinam ferramentas comuns de desenvolvimento em uma única interface gráfica de usuário (GUI). Um exemplo de IDE muito utilizada é o VSCode (*Visual Studio Code*).

​		> __Framework__ : Os chamados Frameworks são facilitadores dos trabalhos dos devs. São ferramentas que trazem diversas soluções pré-definidas, que descomplicam o trabalho, auxiliando a ganhar tempo, automatizando repetições de tarefas, etc. Porém, é recomendado que se conheça a __linguagem__ de cada *framework* antes de utilizá-los.  

Ex: 

- __Angular__: Utilizado no front-end, utilizam as linguagens HTML, SScript, CSS.
- __Laravel__: Utilizado muito por quem utiliza linguagem PHP.
- __VUE__: Utiliza linguagem Javascript
- __Spring__: Muito utilizado no back-end, utiliza a linguagem Java.

​	

- __Back-end__: Trabalha em boa parte dos casos fazendo a "ponte" de ligação entre os dados que vêm do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias em um ambiente onde o usuário final não tenha acesso e possa manipular algo. Utiliza muito as linguagens __Java, PHP, C#__. 

## API

As APIs (*Application Programming Interface*) é uma interface de programação de aplicativos, muito utilizada como uma "comunicação" entre o front e o backend. Pode ser interpretado como um intermediário de software, que permite que dois apps conversem entre si. 

Ex: Enviar uma mensagem através do faceook, verificar previsão do tempo no celular, etc. 



- __Desenvolvedor Fullstack__: É o desenvolvedor que trabalha tanto com backend como frontend. Esses desenvolvedores precisam ter habilidades em ampla variedade de linguagens de programação. 

- __QA (*Quality Assurance*)__: Conjunto de ações que as empresas realizam com o objetivo de entregar aos consumidores um produto ou serviço de alta qualidade. Geram confiança e segurança aos clientes, indicando que os seus produtos terão a qualidade esperada na etapa de implantações. Este, por sua vez, deve conhecer as atividades do projeto, e ter um perfil analítico, verificando os padrões de qualidade, se estão sendo entregues os requisitos esperados. 

  

> Infraestrutura:
>
> Servidor: Nuvens públicas ou privadas.
>
> Servidor Privado: Hardwares específicos. 
>
> Geralmente, são servidores que exigem uma segurança feita por TI (lógica e física), necessita também de mão de obra especializada (software e harware) e uma infraestrutura local.
>
> Servidores públicos: São servidores disponibilizados via internet, ao invés de físico. Geralmente são mais baratos do que servidores privados.
>
> Ex: Alibaba Cloud, Amazon Webservices, Azure, Oracle Cloude, Google Cloud Platform, etc.
>
> As vantagens de se utilizar um servidor público são o preço (paga somente o que usar), a facilidade de contratação, configuração e infraestrutura, a escalabilidade e a performance. 



- __Profissional de Cloud Computing__: É responsável pela estrutura de nuvem oferecidas aos clientes. Escolhe as tecnologias que serão utilizadas, quais operadores são mais interessantes, as peças que serão integradas, e, no fim, cuida do que foi construído. 

### Desenvolvimento Mobile 

- Também utiliza desenvolvedores front e backend.
- Utilizam linguagem Java e/ou Kotlin.
- Geralmente utilizam o Android Studio (para Android) - PC ou Mac.
- Para desenvolvimento IOS utilizam o XCode (Linguagem Swift) - Apenas MacPro ou Hardware Apple.

### Desenvolvimento Híbrido

- Atende Android e Ios, através de *frameworks* e plataformas específicas (Ionic, Flutter, React Native).
- Utiliza Web, sendo assim, frontend utiliza Java, CSS e HTML.
